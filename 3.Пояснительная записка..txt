Пояснительная записка.


Проект "WebServer+API"

Автор: Мифтахова Ева

Идея проекта заключается в том, чтобы разобраться в теме "Сайты", использовать как можно больше доступных библиотек и инструментов, сделать сайт про горные лыжи и вдохновить других людей на это интересное увлечение. 

Информация: LoginForm, функция login - для окошка авторизации пользователя, функции skiing_info()- страница основной информации про горные лыжи, skiing_image()- страница фотографий для поднятия мотивании и вдохновения:)
шаблоны: base.html - страница авторизации, login.html - дополнение к ней, skiing.html - содержание первой страницы, вся информация про горные лыжи, а skiing_image.html - содержание второй страницы с фотографиями.
database.db - база данных с пользователями и их паролями.
models.py - модель для работы с базой данных.
registration.html и register -работа с регистацией пользователя. 

Для запуска нужны библиотеки flask, flask_wtf, wtforms, wtforms.validators, sqlite3, Boostrap, также используется API для меток горнолыжных курортов на карте.
Видео проекта представлено в файле.
